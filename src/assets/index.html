Kepler62 API

Available requests:

  • GET /owners: returns a list of all owners
  • GET /owners/:id: returns a specific owner with the specified ID
  • GET /stores: returns a list of all stores
  • GET /stores/:id: returns a specific store with the specified ID
  • GET /offers: returns a list of all offers
  • GET /offers/:id: returns a specific offer with the specified ID
  • GET /items: returns a list of all items
  • GET /items/:id: returns a specific item with the specified ID
  • GET /craftingtables: returns a list of all crafting tables
  • GET /craftingtables/:id: returns a specific crafting table with the specified ID
  • GET /pricehistory: returns a list of all price history
  • GET /pricehistory/:id: returns a specific price history by the offerId
